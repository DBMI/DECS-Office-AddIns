## Extract Text

![image info](./toolbar.png)

This tool processes HTML data From a REDCap survey, extracting both the patient message and provider's reply out of the surrounding HTML code.

Starting with the HTML-wrapped messages:

![image info](./html_messages.png)

...pressing the `Extract Text` button parses the HTML and creates two new columns for the patient message ant their provider's reply:

![image info](./results.png)

*Please note that the messages shown here are synthetic, created for software testing & development.*
