DECS Word Add-Ins

Creates a custom buttons in Microsoft Word that allows user to:

  • Scan a statement of work file & create SQL code that searches for the ICD-10 codes and names listed in the SoW.
  • Turn a list of MRNs into a SQL snippet that imports the list into a query.

Installation

  • Download the Word Add-Ins folder from Sharepoint.
  • Run setup.exe.

Operation

Extract ICD codes

Sometimes Statements of Work (SoW) contain lists of medical conditions and ICD-10 codes to be reported on. Pressing the Extract ICD button causes the app to scan the open Word document for lines that look like medical conditions and their associated ICD-10 codes.
